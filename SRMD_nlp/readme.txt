# 文本二分类模型集合（SMOTE+LightGBM/CatBoost等）

该代码库包含基于SMOTE过采样技术结合多种机器学习模型（LightGBM、CatBoost等）的文本二分类任务实现，支持数据预处理、模型训练、评估可视化及特征重要性分析。

## 环境依赖
请先安装依赖包：
```bash
pip install -r requirements.txt
数据格式要求
输入数据需为 Excel 文件，格式如下（示例数据md格式，此处为示例数据，实际上应该为医疗文本）：
| 文本内容（第 2 列）| 标签（第 3 列）|
|----|----|
| 这家餐厅的菜品很美味，服务也很周到 | 1 |
| 电影情节枯燥，演员演技差，不推荐观看 | 0 |
|手机续航能力强，拍照效果出色，值得购买 | 1 |
|快递配送延迟，客服态度恶劣，体验极差 | 0 |
| 这本书内容深刻，语言流畅，强烈推荐 | 1 |
说明：
Excel 文件需包含至少 2 列数据（从第 2 列开始，第 1 列会被忽略）
第 2 列为文本内容（待分类的文本）
第 3 列为标签（二分类，0 或 1）
程序会自动过滤掉文本或标签为空的行
主要功能
中文文本分词与预处理（含停用词过滤）
TF-IDF 特征提取
SMOTE 过采样处理类别不平衡问题
多种模型训练（LightGBM、CatBoost 等）
自动寻找最佳分类阈值（以 F1 分数为目标）
模型评估可视化（混淆矩阵、ROC 曲线、PR 曲线）
SHAP 值特征重要性分析（支持中英文展示）
模型保存与复用
输出文件
模型文件（保存于model目录）
评估图表（保存于fig目录）：
混淆矩阵
ROC 曲线
PR 曲线
SHAP 特征重要性图（中英文）
特征翻译对照表
